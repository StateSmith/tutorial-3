# Run StateSmith code gen
Run in this directory:
```
ss.cli run -h
```

<br>

# Retrieve required packages
Use npm to restore the required packages. We just rely on `prompt-sync` to easily get user input.
```bash
npm install
```

<br>

# Run nodejs program
```bash
node index.js
```
